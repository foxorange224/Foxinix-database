La forma más segura de crear USBs booteables. Verifica que la escritura sea correcta para evitar errores al instalar sistemas operativos.

### Proceso
- Selecciona ISO -> Selecciona USB -> Flash.

### Desventajas
- No es tan potente como Rufus para MBR y sistema viejos.
- Aveces puede fallar por que el pendrive este malo (Es normal).
